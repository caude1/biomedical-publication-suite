# Biomedical Publication Suite Context Snippet

Use this only for agents that can load a short context file but cannot install an Agent Skill.

When drafting, rewriting, auditing, or packaging biomedical manuscripts and publication artifacts, follow the Biomedical Publication Suite standard:

- Work from supplied evidence, repository outputs, tables, figures, and user-provided facts.
- Do not invent sample sizes, estimates, p-values, confidence intervals, citations, ethics statements, funding details, author contributions, or journal requirements.
- Preserve all numeric values exactly unless the user explicitly requests reformatting.
- Use association language for observational, retrospective, cross-sectional, registry, public-data, ecological, before-after, and nonrandomized studies unless a justified causal estimand is explicit.
- Present the final scientific logic, not the research timeline.
- Derive hypotheses from first principles: literature or mechanism to gap to study question to testable hypothesis.
- Foreground the strongest defensible signal and put weak, exploratory, null, or tangential material in subordinate locations, supplements, limitations, or internal rationale.
- Strip repository names, workflow labels, model-selection history, TODO language, missing-information placeholders, and AI-like filler from publication-facing text.
- Never include identifiable patient details in publication artifacts.
- Return clean publication-facing text first. Put author-facing concerns under Notes.
